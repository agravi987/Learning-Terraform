# Learning Terraform

Welcome to my Terraform learning journey repository. This project serves as a comprehensive step-by-step guide and practical playground for understanding Terraform concepts, from basics to provisioning cloud infrastructure.

## Repository Structure

- **`1_Basic Concepts/`**: Theoretical foundation and fundamental concepts of Infrastructure as Code (IaC) and Terraform.
- **`2_Setup of Terraform/`**: Instructions and configurations for setting up the local environment and Terraform CLI.
- **`3_HCL/`**: Deep dive into HashiCorp Configuration Language (HCL) syntax, blocks, and structures.
- **`4_EC2_provisioning/`**: Practical example of provisioning AWS resources, including an EC2 instance, Security Groups, and configuring SSH key pairs.

## Getting Started

1. Navigate to any of the subdirectories to explore specific topics.
2. Review the individual `readme.md` files in each folder for detailed, step-by-step learning notes.
